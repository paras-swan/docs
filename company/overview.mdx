## Getting Started

Follow these steps to begin using Swan's powerful AI models.

### Step 1: Sign up for a Console account

To access APIs, you first need to create an account on our Console:

1. Visit Swan's web Console and enter your email.
2. Verify your email address by clicking the link in the confirmation email sent to you.

---

### Step 2: Explore the Console

The Console is your central hub for interacting with Swan's services. It offers various features to help you evaluate capabilities and start developing:

- **Invite Collaborators**: Add new users to your organization and control user access levels in the **Members & Invites** tab within **Settings**.
- **Track Usage and Credits**: Monitor usage metrics for your organization through **Usage Logs**.
- **Manage API Keys**: Generate, disable, and delete your API keys in the **API Keys** section of your Account Settings. These keys are crucial for authenticating API requests and integrating Swan into your application.
- **Access Documentation**: Navigate through comprehensive documentation for everything from model overviews to advanced usage patterns.

---

### Step 3: Generate an API Key

To develop with Swan, you’ll need an API key. Follow these steps to create one:

1. Log in to the Console.
2. Navigate to the **API Keys** section within your Account Settings.
3. Click the **Create New API Key** button on the top right.
4. Provide a descriptive name for your key (e.g., “My_First_SWAN_App”) and click **Create API Key**.
5. Copy the generated API key and store it securely. **Note**: You won’t be able to view the key again after leaving the page, so make sure to save it in a safe location.
